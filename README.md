## Install

```bash
yarn add @splinetool/r3f-spline @splinetool/loader
```

or

```bash
npm install @splinetool/r3f-spline @splinetool/loader
```

**NOTE**: make sure you also install the `@react-three/fiber` and `three` dependencies.
